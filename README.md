# Exploratory data analysis with napari, scikit-image, and Jupyter
## Halfway to I2k 2025

This site contains materials for the Halfway to I2K 2025 workshop: "Exploratory data analysis with napari, scikit-image, and Jupyter", Nov 18, 2025.

This workshop will focus on using napari from a Jupyter notebook to carry out exploratory image analysis. 
You will learn how to launch napari from Jupyter, how to interact with it, and how to document your work using nbscreenshot.
